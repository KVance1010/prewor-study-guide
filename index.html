CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Value Selectors
    • Selector: element[attr] Example: a[href="https://example.com"] Description:exactly matches
    • Selector: element[attr~= value] Example: p[class~="special"] Description: exactly value or contains value in a list
    • Selector: element[attr|=value] Example: div[lang|="zh"] Description: exactly value or begins with value immediately followed by a hyphen

  • Substring Selectors
    • Selector: element[attr^= value] Example: li[class^="box-"] Description: value begins with
    • Selector: element[attr$= value] Example: li[class$="box-"] Description: value ends with
    • Selector: element[attr*= value] Example: li[class*="box-"] Description: value contains the value anywhere

  • Combinators
    • Descendant Combinators: " "(space) Example: div span Description: more specific element selection
    • Child Combinators: > Example: ul>ui Description: selects nodes that are the direct children of the element
    • General Sibling Combinator: ~ Example: p ~ span Description: will match all elements that follow the first element
    • Adjacent Sibling Combinator: + Example: h2 + pDescription: matches the second element only if it immediately follows the first element
    • Column Combinator: || Example: col || td Description: will match all td elements that belong to the scope of the column
  • Pseudo-classes and Pseudo-elements
    • Pseudo class: : Example: a:visited Description: select elements based on a state (:hover, :fullscreen, :modal, :picture-in-picture, :first-child, :autofill, More Examples
    • Pseudo Element: :: Example: p::first-line Description: will match all specified element. (every first line of a p element)
    • Pseudo-classes and Pseudo-elements can be combined. Example: article p:first-child::first-line{}
    • Content can be added to HTML with CSS: Example: .box::before{content: "this will be added to the html page."} Example: .box::before or .box::after
    • Flex-box and Grid
    • Flex-box if good for single column organization
    • Grid allows you to layout the page horizontally and verticality

JavaScript

  • Split can be used to break up a string Example: stringV = "John,Pete,Harry" stringV.split(",", 3) (break point, how many words) no value is more common for the 2nd part
  • Substring can be used to break up a string: (start index, end index). Substring and Slice are the same. Slice can use neg numbers to start from the end
  • myText.replace('Kyle', 'John') ... or replaceAll
  • Join can be used to make a string out of an array. Split is a string to an array.
  • Default parameters. Example: hello(name = 'chris') specifying a default values
  • "truthy" and "falsy". falsy is 0, on, "", null, undefined, and NaN.
